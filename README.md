# max-arias

Personal consulting website for Maximiliano Arias.

## Stack

- Astro 7
- Tailwind CSS 4
- Static output for Cloudflare Pages

Deploy `dist/` to Cloudflare Pages, or connect the repo with build command `npm run build` and output directory `dist`.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
